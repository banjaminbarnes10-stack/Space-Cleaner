Space Cleaner - аркадная игра для Android, написанная на Java с использованием LibGDX и Box2D. Игрок управляет космическим кораблём касанием экрана. Корабль автоматически стреляет по космическому мусору, а сложность постепенно увеличивается за счёт сокращения интервала появления препятствий. В проекте реализованы главное меню, экран настроек и игровой экран, три жизни, очки, пауза, таблица пяти лучших результатов, музыка и звуковые эффекты. Настройки музыки, звука и рекорды сохраняются в постоянной памяти.
Запуск на компьютере:
```powershell
.\gradlew.bat :desktop:run
```
Сборка Android APK:
```powershell
.\gradlew.bat :android:assembleDebug
```
APK создаётся в папке:
```text
android/build/outputs/apk/debug/android-debug.apk
```
Структура:
`core` содержит игровую логику, экраны, физические объекты и UI-компоненты.
`android` отвечает за запуск игры на Android.
`desktop` используется для проверки игры на компьютере.
`assets` содержит текстуры, шрифт, музыку и звуковые эффекты.
